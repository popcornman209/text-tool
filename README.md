# text-tool
a little text tool for python

===== syntax =====<br />
~~         : ~<br />
~v(n)      : sets visual (see below)<br />
~c         : clear<br />
~d(t)      : delay(time)<br />
~s(s,v)    : setting(name, value)<br />
~i         : press enter to continue<br />
~p(s)      : executes string (python)<br />

===== settings =====<br />
type_speed : delay between letters<br />
